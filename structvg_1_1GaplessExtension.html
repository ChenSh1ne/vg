Detailed Description

A result of the gapless extension of a seed.

  • 'path' covers 'core_interval' of the read.
  • The intervals are semi-open [first, second) intervals.
  • 'state' is the search state corresponding to 'path'.
  • If 'is_full_alignment' is set, the path is a full-length alignment and may contain mismatches. Otherwise it is a maximal unambiguous exact extension of a seed.
  • If the path is not a full-length alignment, 'flanked_interval' is a read interval extending 'core_interval'. The extension may contain mismatches, but it starts and ends with a match.
  • The mismatching read positions are stored in 'mismatch_positions' in sorted order.
