Detailed Description

A HandleGraph implementation that wraps some other handle graph and reverses but does NOT complement the sequences.

See also: BackwardsGraph

TODO: we inherit publically from BackwardsGraph, because inheriting it privately and publically inheriting HandleGraph again produces a warning. DO NOT use this as a BackwardsGraph; it does a different thing.
