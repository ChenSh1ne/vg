Detailed Description

A class that supports haplotype-consistent seed extension using GBWTGraph. Each seed is a pair of matching read/graph positions and each extension is a gapless alignment of an interval of the read to a haplotype. A cluster is a vector of distinct seeds. The cluster may initially be in an arbitrary order, but it will be sorted during the extension. All seeds in a cluster should correspond to the same alignment or positions near it.
